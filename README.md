# Getting Started

To get started with the BulletFarm API, follow these instructions to set up your development environment and run the server locally.

### Prerequisites
- Go 1.16 or later installed on your machine.
- Git installed for version control.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/grahamprimm/bulletfarm-test-repo-beta.git
   cd bulletfarm-test-repo-beta
   ```
2. Install the necessary Go packages:
   ```bash
   go mod tidy
   ```

### Build and Run
To build the Go server, run the following command:
```bash
go build -o bulletfarm
```

To run the server, use:
```bash
./bulletfarm
```
The server will run on `http://localhost:8080`.
